Патч для редактора карт v3.3

Патч обеспечивает поддержку новых монстров, артефактов и т.д. и добавляет новые возможности: изменение вида объектов, копирование части карты из одного места в другое, связывание событий и т.д.
Английская версия
Русская версия


Справка для редактора карт v1.2

Обновленная справка, содержащая описание всех вещей, добавленных в ВоГе.
Английская версия
Русская версия


32BitPatch

Позволяет запускать Героев в оконном режиме при любом качестве цвета, установленном в системе. Патч может быть установлен на любую версию Героев 3, включая Хроники.
(c) Bourn и я.


Улучшенные цвета интерфейса

Затемняет цвета игроков, в результате чего интерфейс выглядит гораздо лучше. Распакуйте этот маленький архив в папку Data.


Новые боевые сетки

Положите понравившийся вариант в папку Data. Я советую третий.


Синхронизация звуков существ

Некоторые существа издают звуки с запазданием. Распакуйте архив в папку с игрой, чтобы это исправить.


MMArchive v1.2.1

Универсальный редактор архивов M&M и Heroes 3. Поддерживает все типы архивов, кроме *.hwl.

Избранное

Избранное для h3sprites.lod и h3bitmaps.lod с рассортированным списком файлов. Русская версия. Было бы очень хорошо, если бы кто-то сделал английскую. Автор — Black Phantom, с небольшими моими правками.

LodCompare v1.3.1

Устарел, т.к. включён в MMArchive. Сравнивает 2 архива lod/vid/snd и показывает список измененных файлов. Позволяет создать новый архив, содержащий только эти файлы. Это более эффективный способ распространения модов: распространять архив с измененными файлами, как свой LOD, загружаемый моим патчем, или добавлять его к архиву пользователя с помощью LodMerge. Исходники (Delphi 2006)


LodMerge v1.1.1

Позволяет соединить 2 архива lod/vid/snd. Исходники (Delphi 2006)


SimpleLod v1.3

Позволяет легко добавлять файлы в .lod-ахривы Heroes 3. Исходники (Delphi 2006)


Def Preview v1.2.1

Показывает файлы .def, позволяет извлекать кадры и готовить деф-лист для Def Tool.


Def Tool v3.4.2

Создает файлы .def. Еще может обрабатывать картинки для интерфейса Героев.

Def Tool v2.1.1

Старая версия. Вам могут пригодиться кнопки Соединить и Разъединить. С помощью них можно привести кадры к палитре более качественно.


Txt Edit v1.3.1

Редактор текстовых таблиц Heroes 3 и M&M.


RSBinkPlayer v1.2.5

Просмотрщик файлов .bik и .smk. Умеет просматривать геройские .bik-файлы.


Fnt Editor

Старая программа для редактирования шрифтов Heroes 3 и M&M.


Редактор диалогов для ресивера !!DL

Ресивер !!DL позволяет Вам создавать свои диалоги для версий ВоГа TE, Era и 3.59. Справки нет, т.к. развитие программы прекращено. Новая версия может когда-нибудь выйти, но только для новой системы диалогов WoG 3.59.


Менеджер портретов

Позволяет легко менять портреты героев.


Mon Power

Вычисляет силу монстров. Результат – значения, близкие к AI Value и Fight Value. Учитывает только базовые параметры (игнорирует скорость, стрельбу и т.п.).
